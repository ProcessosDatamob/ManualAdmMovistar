# INTRODUCCIÓN

El propósito de este manual es ayudar al usuario administrador en el uso y configuración del **Movistar Gestión de Dispositivos.**

> Este software cumple íntegramente con la Ley General de Protección de Datos (LGPD), garantizando la seguridad y el cumplimiento en el tratamiento de la información.
